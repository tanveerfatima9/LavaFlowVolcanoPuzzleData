Ge dig ut på ett eldigt äventyr i LavaFlow, det ultimata vulkaniska pusselspelet som utmanar ditt strategiska tänkande och problemlösningsförmåga. Ta kontroll över smält lava som bryter ut från en mäktig vulkan och styr dess flöde genom rutnätsbaserade pussel för att nå målet destinationer som byar, forntida altare och blommande trädgårdar som väntar på vulkanens välsignelse. Ta dig igenom fem unika nivåer med ökande svårighetsgrad, där alla har olika layouter, hinder och mål. Använd en mängd olika strategiska rörelement för att kontrollera lavan, inklusive raka rör för att hålla flödet i rörelse, hörnrör för att vrid den i rät vinkel, T-delade rör för att dela flödet, korsande rör för att skapa komplexa korsningar och blockerande element för att stoppa oönskade vägar. Tävla mot tiden när lavan svalnar när du färdas genom tomma utrymmen, vilket kräver noggrann placering av element för att hålla den i schack flytande. Tjäna upp till tre stjärnor per nivå genom att slutföra etapper effektivt, nå alla mål och använda så få element som möjligt. samtidigt som du spårar dina framsteg, bästa tider och prestationer med detaljerad statistik. Fördjupa dig i ett vackert mörkt tema med djupblå toner och glödande lavaeffekter. För att spela, dra rörelement från den nedre panelen till rutnätet, rotera dem med ett dubbeltryck, skapa en bana från vulkanen till målen, tryck sedan på avfyrningsknappen och titta på lavaflödet. Nå det erforderliga antalet mål för att slutföra varje nivå. Perfekt för pusselentusiaster, strategiälskare och alla som söker en unik hjärngymnastikupplevelse, LavaFlow inbjuder dig till Bemästra konsten att vulkanisk ingenjörskonst och släpp lös vulkanens kraft.
